### Task-1

Loop through an object and print only `numeric values`

**Input:**
<br>

let myObject = {

    name: 'John Doe',
    age: 25,
    city: 'Example City',
    isStudent: true

};

**Output:**

<br>
    age : 25

### Tasks-2:

Print the `keys` of the following object :

**Input:**
<br>
var myObject = {

        name: 'John Doe',
        age: 25,
        city: 'Example City',
        isStudent: true

};

**Output:**
<br>
[ 'name', 'age', 'city', 'isStudent' ]
